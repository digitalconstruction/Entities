Abstract

This Entities ontology is a module of Digital Construction Ontology. The purpose of this module is to support data sharing of the object information that related to the construction project. This module describes the concepts (or ‘classes’) of basic objects in the construction process and their properties and hierarchy. The core concept is the ‘Identifiable Entity’, which refers the top abstract concept of all the objects that has its own identifier and classification that classified by some classification scheme.
